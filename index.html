It started with a
Windows Vista.

The First Computer

Before I had a real computer, I had an old Windows Vista. I spent a lot of time just poking around — figuring out what file extensions meant, watching YouTube, learning how things worked. Minecraft on PS4 was my first real game. It made me want to understand what was behind everything.

My Dad Said: Here's Python

My dad signed me up for WhiteHat Jr when I was around 9. I built Pong, brick breaker, small games that actually ran. I really liked it. That led to Unity — two Udemy courses in C# 2D and 3D by age 11. I built a lot of prototypes that never went anywhere. That was fine.

High School, Then Everything Else

At 13 I published my first Minecraft mod on CurseForge. I built an AI Snake agent in Python. Then high school started and I launched Arvex Games — a full web gaming platform built from scratch, with zero web experience. That one taught me the most. It's discontinued now, but it led directly to modrift, Rift Loader, and Alloy Editor.
