It started with a
Windows Vista.

The First Computer

Before I had a real computer, I had an old Windows Vista. I spent hours watching YouTube, learning what file extensions meant, figuring out how things worked. Minecraft on PS4 was my first real game — and the beginning of an obsession with systems and how they're built.

My Dad Said: Here's Python

My dad signed me up for WhiteHat Jr when I was around 9-10. I got certified. I built Pong, brick breaker, little games that actually ran. I loved it more than anything I'd tried before. Naturally, I aimed bigger. If I could build games in Python, why not Unity?

Two Udemy Courses Later

At 10-11, I finished C# Unity 2D and 3D courses on Udemy. Then I built game after game. An AI Snake agent. My first Minecraft mod on CurseForge. High school arrived and instead of slowing down, I launched Arvex Games, then modrift, then Rift Loader, then Alloy Editor. The builds kept coming. They still do.
