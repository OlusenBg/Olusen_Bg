AI Snake Agent

A snake that learned from every mistake until it didn't make any.

Local Project

Teaching a snake to think

Built in the summer before high school. The concept was simple: create a Python agent that could play Snake, and let it learn through trial and error until it got good. The implementation was anything but simple for someone exploring machine learning for the first time.

The agent starts knowing nothing. It collides with walls, runs into itself, misses food that's right next to it. But with every game, it accumulates experience. Reward signals tell it what worked and what didn't. Gradually, game by game, the snake gets smarter. It learns to avoid walls. It learns to chase food. Eventually, it stops making the mistakes it made a thousand games ago.

This was my first hands-on experience with reinforcement learning concepts — reward systems, state representation, algorithm-driven behavior. Less about the math, more about the ideas behind why it works.

How the agent improves

State representation

The agent perceives its environment: where the walls are, where the food is, where its own body is. State is the input that drives decisions.

Reward signal

Eating food is rewarded. Dying is penalized. The agent learns to maximize reward over time — simple rules, complex emergent behavior.

Trial and error at scale

The agent plays hundreds of games. Early games are chaotic. Over time, patterns emerge. Good moves get reinforced. Bad moves get pruned.

Convergence

Eventually, the agent's behavior converges to something that actually works — consistent food collection, wall avoidance, self-preservation.
