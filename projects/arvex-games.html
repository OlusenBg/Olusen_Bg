Arvex Games

My most ambitious project. Built from zero web knowledge. And it worked.

A full platform. Built by a kid who didn't know web dev yet.

Arvex Games was a full ad-free web gaming platform — ArvexGames.com — launched in 9th grade. I had no prior web development experience. No tutorials for this specific thing. Just a goal and the willingness to break things until they worked.

The platform featured curated browser games — ULTRAKILL, PolyTrack, Hypersomnia, and more — alongside a full user account system with profiles, a virtual coin economy with daily streaks, leaderboards, progression systems, social features (friends, chat), and a developer submission program where indie game creators could submit their games for hosting.

No frameworks handed me these features. I learned hosting, domains, auth systems, virtual economies, moderation tools, Cloudflare R2 buckets, and TypeScript by building them.

What was built

Virtual Coin Economy

Daily streaks, earned currency, and progression systems. Users had real reasons to come back — and the system held up.

Full Auth System

User accounts, profiles, sessions. Built without a framework spoon-feeding me the implementation. Every bug taught me something.

Social Features

Friends, chat, leaderboards. Not just a static game list — an actual community platform that people could use together.

Dev Submission Program

Indie developers could submit their games for hosting on the platform. Moderation tools, review process, Cloudflare R2 storage.

// WHAT I LEARNED

"I started this with zero knowledge of web development. No frameworks, no tutorials, just a goal and a willingness to break things until they worked. Arvex Games taught me more than any course ever could."

It's discontinued now, for personal reasons. But it remains my most complete platform build — and the thing that proved to me I could teach myself anything.
