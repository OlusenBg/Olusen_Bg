The problem with modding

Every game has its own modding ecosystem. Different tools, different setups, different ways things break. Minecraft wants Java and Gradle. Skyrim wants the Creation Kit. Cyberpunk 2077 wants something else entirely. Before you've written a single line of mod logic, you've already fought a dozen install issues.

modrift flips this. Instead of fighting your local environment, you open a browser tab and start building. The platform handles the context — which game you're targeting, which APIs exist, which patterns work. The AI coding assistant (powered by Claude) fills in the gaps. Clerk handles auth so your mods are tied to your account. Neon manages the database. Cloudflare Workers runs the backend at the edge.

This is the platform I wish had existed when I started modding.

Key features

AI Coding Assistant

Powered by the Claude API. It knows the modding APIs for each supported game and helps you write correct, idiomatic mod code — fast.

Cross-game support

Minecraft, Skyrim, Cyberpunk 2077, Terraria, and more. One platform, consistent workflow across every game you want to mod.

Browser-based editor

No local setup. Open modrift.dev, sign in, and build. Your mods live in the cloud and sync across devices.

Cloudflare-powered backend

Cloudflare Workers and Pages keep everything fast and at the edge. Neon DB handles persistence. Clerk manages auth end-to-end.
